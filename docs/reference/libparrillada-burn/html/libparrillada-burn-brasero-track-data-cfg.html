parrillada_track_data_cfg_span ()

ParrilladaBurnResult   parrillada_track_data_cfg_span         (ParrilladaTrackDataCfg *track,
                                                         goffset sectors,
                                                         ParrilladaTrackData *new_track);

Creates a new ParrilladaTrackData (stored in new_track) from the files contained in track. The sum of their sizes does not exceed sectors. This allows to burn a tree on multiple discs. This function can be called repeatedly; in this case if some files were left out after the previous calls, the newly created ParrilladaTrackData is created with all or part of the remaining files.

track :

 a ParrilladaTrackDataCfg

sectors :

 a goffset

new_track :

 a ParrilladaTrackData

Returns :

 a ParrilladaBurnResult. PARRILLADA_BURN_OK if there is not anymore data. PARRILLADA_BURN_RETRY if the operation was successful and a new ParrilladaTrackDataCfg was created. PARRILLADA_BURN_ERR otherwise.
